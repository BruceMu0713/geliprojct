### 格力:mall.gree项目说明


## 功能

1. 基本结构与样式（5个html页面+sass样式），结构，样式，行为分离
2. 二级导航(tab切换)
3.楼梯
4. 数据渲染
    - 主页数据渲染
    - 商品详情数据渲染
    - 购物车数据渲染
5. 图片懒加载(jquery.lazyload)
6. 密码加密（php sha1）
7. 登录、注册数据请求，正则验证
8. 轮播图，走马灯
9. 商品选项卡
10. 登录后主页用户名显示（获取localStorage），点击退出（删localStorage）
11. 购物车商品数量添加，删除，对应总价加减
12. 购物车点击删除（删除cookies），删除商品
